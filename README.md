# Plataforma de cursos en linea 
Plataforma web educativa que permite a instructores crear y gestionar cursos en línea.

## Descripcion del proyecto

Sistema desarrollado como proyecto final del curso de Ingeniería de Software que implementa:
- Gestión completa de cursos
- Sistema de autenticación de usuarios
- Inscripción de estudiantes
- Búsqueda y filtrado de cursos
